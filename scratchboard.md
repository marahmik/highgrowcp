# 쌀체듈러 (SSalcheduler) 기획 및 기능 정의서

초기 기획안(`docs/brainstorming/scratchboard.md`)과 최근 진행된 업데이트 내용을 종합한 최신 요구사항 및 기능 명세서입니다.

## 📌 핵심 목적
- 매장 파트타임/직원들의 월별 근무 스케줄을 효율적으로 관리하기 위한 웹 애플리케이션.
- 각 사용자(관리자, 직원 등)가 직접 로그인하여 권한과 직급에 따라 스케줄을 입력하고 관리할 수 있도록 구성.

## 👥 권한 및 주요 기능

### 1. 일반 사용자 (직원)
- **가입 및 매장 합류**: 이메일 또는 구글 계정으로 가입 후, 생성된 매장에 가입 요청을 보냄.
- **개인 스케줄 등록/수정**: 관리자 승인 후 소속된 매장 페이지에 입장하여, 월별 달력에 본인의 스케줄을 등록(근무/휴무 상태 선택).
- **마이페이지**: 가입된 매장 목록 확인 및 개별 매장 페이지 이동.

### 2. 관리자 (대표자 / Supervisor)
- **회원 관리 (멤버 관리)**:
  - 가입 요청 승인 및 권한 변경.
  - 멤버 이름 닉네임처럼 변경(Rename) 가능.
  - '매장 떠나기(Leave store / Transfer)' 와 '영구 삭제(Permanent delete)' 기능 분리 제공.
- **매장 관리**:
  - 다중 매장 생성 및 정보 수정 가능.
  - 매장별로 관리자만 작성 및 확인 가능한 '메모(Memo Box)' 기능.
- **통합 스케줄 관리 (기존 승인 탭 대체)**:
  - 복잡한 '결재/승인(Approvals)' 탭을 없애고 관리자 편의성을 위해 **[통합 캘린더 (All Schedules)]** 기능 도입.
  - 전체 매장의 스케줄을 한눈에 보고 필터링하여 관리자 권한으로 직접 수정 및 삭제.
  - 슈퍼바이저(관리자) 뷰에서는 할당되지 않은 인원(Ghost slots)을 숨기거나 보이게 하는 커스텀 옵션 제공, 멤버 카운트 정렬 기능 지원.

## 📅 달력 및 스케줄 상태 정의
1월부터 12월까지 월 단위로 조회 및 작성. 요일, 근무 형태, 휴무일 등은 한눈에 구분이 쉽도록 직관적인 배경색으로 구분됨.

- **근무 형태**: `오픈`, `미들`, `마감`, `종일`
- **휴무 종류**: `연차`, `반차`, `대체휴`, `병가`, `요청`

## 🖥️ 페이지 구성

1. **메인 페이지 (Main Page)** 
   - 서비스 소개 및 로그인/회원가입.
   - 글로벌 워터마크/푸터 (HDH Hugo Kim & Jason Kim) 표시.
2. **마이 페이지 (My Page)**
   - 일반 직원이 로그인 후 마주하는 화면. 
   - 매장 목록 확인, 소속 매장 진입 혹은 미가입 시 매장 가입 요청.
3. **매장 페이지 (Store Page)**
   - 소속 인원들의 월별 달력이 표시되는 메인 작업 화면.
   - 다양한 기기 환경에서 깨짐이나 좌우 스크롤(가로 스크롤) 불편함이 없도록 **반응형(Adaptive) 디자인** 적용 (데스크톱 환경 포함, 한 화면에 1일~말일 출력 완료).
4. **관리자 페이지 (Admin Page)**
   - 관리자 권한을 가진 유저가 접근하는 통합 관리 대시보드.
   - **회원 관리**, **매장 관리**, **통합 캘린더(스케줄 전권 수정 및 뷰어)**로 구성.
   - ⚠️ **[업데이트] 통합 캘린더 개편**: 여러 매장에 겹치게 가입된 인원의 스케줄이 중복으로 나오는 기존의 방식을 폐기하고, 멤버 ID를 기준으로 완전히 하나로 합쳐서 보여줍니다. (예: `Haram Kim (송도/인천/전체관리자)`)
   - 각 점포별 매장 메모는 캘린더 하단에 티커별로 분리해서 개별 저장/확인할 수 있도록 재배치되었습니다.
